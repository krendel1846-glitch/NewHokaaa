# Golden WOK Delivery MVP

Статический mobile-first PWA-прототип по техническому заданию.

## Как открыть

Откройте файл `index.html` в браузере.

## Что реализовано

- каталог блюд с категориями, фото, КБЖУ, бейджами и модификаторами;
- корзина с количеством, промокодом, баллами и оформлением заказа;
- демо-трекинг заказа с ETA, статусами и чек-листом;
- профиль, адреса, бонусы, реферальный код и история заказов;
- кухонный экран KDS с чек-листом сборки и статусами;
- админ-панель с KPI, меню, промокодами и ABC-блоком;
- PWA-манифест, иконки, service worker и офлайн-кэш основных файлов.

## Ограничения MVP

Платежи bePaid/ЕРИП, SMS OTP, Telegram-бот, серверная база данных и реальные push-уведомления представлены как интерфейсные сценарии. Для боевого запуска нужно подключить backend, PostgreSQL, Redis и внешние API.
